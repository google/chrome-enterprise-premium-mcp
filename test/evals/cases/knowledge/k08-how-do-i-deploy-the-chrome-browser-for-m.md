## Prompt

How do I deploy the Chrome browser for my organization?

## Golden Response

Download the Chrome Enterprise bundle (MSI, PKG, and policy templates). Deploy via SCCM, Intune, or other MDM tools, and sync management to the cloud via Chrome Enterprise Core. For more details, see the [Download and deploy Chrome Browser](https://support.google.com/chrome/a/answer/7650032) Help Center article.
