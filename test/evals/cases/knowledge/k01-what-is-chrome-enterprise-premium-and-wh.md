## Prompt

What is Chrome Enterprise Premium, and what capabilities does it offer?

## Golden Response

Chrome Enterprise Premium (CEP) is an advanced security offering that builds upon the standard Chrome browser and Chrome Enterprise Core management. It enables secure, Zero Trust access to applications and resources without requiring a VPN. Core capabilities include Data Loss Prevention (DLP), real-time URL and file scanning for malware/phishing, and context-aware access controls for web applications. You can find more information at the [Chrome Enterprise Premium documentation hub](https://cloud.google.com/beyondcorp-enterprise/docs).
