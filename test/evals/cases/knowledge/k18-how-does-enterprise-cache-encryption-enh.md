## Prompt

How does Enterprise Cache Encryption enhance security?

## Golden Response

Enterprise Cache Encryption (under Cache Encryption), enhances security by encrypting local browsing data at rest — including the HTTP cache, cookies, history, and saved passwords in Chrome. This prevents unauthorized access to potentially sensitive data if a device is compromised, lost, or stolen.
