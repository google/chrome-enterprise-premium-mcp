## Prompt

How does Enterprise Cache Encryption enhance security?

## Golden Response

Enterprise Cache Encryption enhances security by encrypting the local HTTP cache
data in Chrome at rest. This prevents unauthorized access to potentially
sensitive corporate content cached by the browser if a device is compromised,
lost, or stolen.
